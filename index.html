Projects

TextMD

Hackathon application aimed to bring medical tools to impoverish areas of the world, where Internet is difficult to access, by using texting. It won the Grand Prize at BigParser's Summer Hackathon.

#TrendingInMedicine

Project started by a couple of my friends and me during HackTJ 2017. It grabs each month's trendy words from a group of medicine journals and displays them as hashtags on the Trending Page much like Twitter.

WolframAlpha Chrome Extension

A side project developed during my 2017 Web Development course using the Wolfram API.

NLTK based RAKE ALGORITHM

Same algorithm used to detect trends from #TrendingInMedicine but made for made more for broader usages. It is able to extract the key phrases and identify topics in any document.
